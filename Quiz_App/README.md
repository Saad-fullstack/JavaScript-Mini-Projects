# 🧠 JavaScript Quiz App

A modern and interactive Quiz Application built with **HTML**, **CSS**, and **Vanilla JavaScript**. This project tests users with multiple-choice questions, tracks their score, includes a countdown timer for each question, and provides a detailed review after the quiz.

This project was created to practice **DOM Manipulation**, **Events**, **Arrays**, **Timers**, and overall JavaScript logic without using any external libraries or frameworks.

---

## 🎮 How to Play

1. Click **Start Quiz**.
2. Read the question carefully.
3. Select one option.
4. Move between questions using **Next** and **Previous**.
5. Every question has **30 seconds**.
6. If the timer expires, the quiz automatically moves to the next question.
7. After the final question, your result is displayed along with a detailed review.

## 🐞 Known Limitations

- Questions are stored directly inside the JavaScript file.
- Quiz progress is not saved after refreshing the page.
- Timer resets when moving between questions.
- No negative marking.
