# 🎨 Color Changer

A simple JavaScript mini project that changes the background color of the webpage when the user clicks on different color buttons.

## 📌 About

This project was built to practice JavaScript DOM manipulation and event handling. Each colored button listens for a click event and updates the page's background color dynamically.

## 🚀 Features

* Change the page background color with a single click.
* Clean and beginner-friendly JavaScript code.
* Uses `querySelectorAll()` to select multiple elements.
* Uses `addEventListener()` to handle user interactions.
* Demonstrates basic DOM manipulation.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Run

1. Clone the repository.
2. Open the `colorChanger` folder.
3. Open `index.html` in your browser.
4. Click on any color box to change the page's background color.
