# ⏰ Digital Clock

A simple and responsive **Digital Clock** built with **HTML, CSS, and JavaScript**. This project displays the current time and date, updating automatically every second.

It was created as a JavaScript practice project to strengthen my understanding of the **Date object**, **DOM manipulation**, and **setInterval()**.

## ✨ Features

* Displays the current time in real time.
* Shows the current date in **DD-MM-YYYY** format.
* Updates automatically every second.
* Responsive and clean user interface.
* Beginner-friendly project with simple and readable code.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📚 What I Practiced

While building this project, I practiced:

* Selecting DOM elements with `querySelector()`
* Working with the `Date` object
* Displaying the current time using `toLocaleTimeString()`
* Extracting day, month, and year
* Formatting numbers with `padStart()`
* Updating the DOM using `innerHTML`
* Running code repeatedly with `setInterval()`
* Using template literals

1. Clone this repository.

```bash

```

3. Open `index.html` in your browser.

That's it! The clock will start automatically.
