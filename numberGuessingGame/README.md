# 🎯 Number Guessing Game (Version 1)

A simple Number Guessing Game built using **HTML**, **CSS**, and **JavaScript**. This project was created to practice JavaScript fundamentals such as DOM manipulation, event handling, functions, conditionals, and game logic.

## 📖 About

The game generates a random number between **1 and 100**.

Your goal is to guess the correct number within **10 attempts**. After each guess, the game tells you whether your guess is:

- 📈 Too high
- 📉 Too low
- ✅ Correct

If you use all 10 attempts without guessing correctly, the game ends and reveals the correct number.

---

## ✨ Features

- Random number generated between 1 and 100
- Input validation
- Displays previous guesses
- Shows remaining attempts
- Feedback after every guess
- Game Over message when attempts are exhausted
- Option to start a new game

```bash

```

3. Open `index.html` in your browser.

No installation or additional packages are required.

---

## 🎮 How to Play

1. Enter a number between **1 and 100**.
2. Click the **Submit** button.
3. Read the feedback.
4. Keep guessing until you find the correct number or run out of attempts.
5. Click **New Game** to play again.
